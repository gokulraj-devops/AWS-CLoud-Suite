ℹ️ About Security Audit
This module runs Prowler — an open-source AWS security scanner — using your login credentials across all enabled regions.
It runs a focused set of ~45 high-value checks covering IAM, S3, EC2, RDS, CloudTrail, GuardDuty, KMS, VPC, and more.

⚡ Auto-Install: If Prowler is not installed, clicking Run Prowler Scan will automatically install it into your virtual environment.

Manual install: source venv/bin/activate && pip install prowler

⏱ Duration: ~5–20 minutes depending on how many AWS regions are enabled in your account.
